# Tailwind UI Audit Skill

1. Run `npx tailwindcss-cli lint` across the project source.
2. Review the resulting output against modern responsive design standards.
3. Provide an executive summary of non-compliant classes.
4. If auto-editing is enabled, optionally apply suggested standard classes.
